# Toad-Vue
> Built by NodeGG and Friends

## Introductions
A Collection of Our Most used Components put together in a simple package.


## How to use package?
```
npm i toad-Vue

or

yarn add toad-vue
```

Link to pages you need

ex.
```
toad-vue/components/Header
```

- Toolbar
